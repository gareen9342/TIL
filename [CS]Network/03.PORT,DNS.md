## PORT

패킷 내에는 출발지 PORT, 목적지 PORT가 있다.    
같은 IP내에서 포트를 이용해 프로세스를 구분할 수 있다. 통신할 애플리케이션을 구분하기 위해서 사용    

0 ~65535의 포트를 할당 가능하며,    
0 ~ 1023 포트는 잘 알려진 포트, 사용하지 않는 것이 좋다.    

- **FTP** : 20, 21    
- **TELNET** : 23    
- **HTTP** : 80    
- **HTTPS** : 443
