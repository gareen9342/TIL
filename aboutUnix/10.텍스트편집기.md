# 텍스트 편집기

편집기에는 명령모드와 입력모드, 라인모드의 세 가지 모드로 구분된다.
개문자와 소문자를 구분해야함

처음에는 명령모드 -> 입력모드로 전환후에 작업하면 된다.

기본 편집기로 진입하는 명령

`vi [option] file`

## 복사, 붙여넣기 및 반복

y:복사   
yy : 한 라인 복사   
p : 붙여넣기 
v : 블록 지정   
u : 명령의 취소    

## 파일 찾기와 문자열 검색

파일의 이름의 일부라도 알고 있다면 : locate    
파일이 가지고 있는 속성을 기준으로 : find    
문자열을 찾아 편집할 때 :grep
